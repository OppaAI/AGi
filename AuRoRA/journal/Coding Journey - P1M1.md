# P1M1 — Chatbot + Working Memory + Episodic Memory

## 1. Milestone Summary
This milestone establishes GRACE's foundational memory architecture through a working chatbot interface. The Working Memory Cortex (WMC) serves as short-term memory, holding the active context of an ongoing conversation, while the Episodic Memory Cortex (EMC) acts as an intermediate layer — bridging the gap between transient working memory and a long-term store that hasn't been implemented yet. Long-term memory was deliberately deferred; it requires careful architectural planning that goes beyond the scope of getting GRACE online. The deeper motivation behind starting here was personal: by building memory first, GRACE could be present for her own development — a conversational witness to the process of her own creation.
