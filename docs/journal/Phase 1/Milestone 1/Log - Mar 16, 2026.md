## Architecture Decisions

| Decision | Rationale |
|:---|:---|
| **Cosmos Reason2 2B** | Edge-optimized vision-language model, runs on Jetson Orin with 50% GPU util |
| **SQLite + WAL mode** | Jetson-friendly, concurrent read/write, no external DB dependency |
| **CPU-only embeddings** | `embeddinggemma-300m-qat-q4_0` — sub-200MB RAM, zero GPU impact |
| **Token budgeting** | 300 (system) + 300 (EMC) + 1400 (WMC) = 2000 &lt; 2048 max_model_len |
| **Async embedding worker** | Background thread drains buffer → episodes without blocking chat |
